# 行動規範

## 目的

このプロジェクトは、誰もが安心して参加できるオープンでウェルカムなコミュニティを目指しています。すべての参加者は、この行動規範を尊重することが期待されます。

## 行動基準

**推奨される行動**:
- 他の参加者に対して礼儀正しく、親切に接する
- 建設的なフィードバックを行い、受け入れる
- コミュニティ全体の利益を優先する
- 異なるバックグラウンドや視点を尊重する

**許容されない行動**:
- ハラスメント、差別、侮辱的な発言
- プロジェクトと無関係な政治・宗教的な議論
- 個人情報の無断公開
- その他、合理的にプロフェッショナルでないとみなされる行動

## 責任範囲

プロジェクトのメンテナーは、この行動規範の基準を明確にし、違反に対して適切な対応を取る責任があります。

## 適用範囲

この行動規範は、プロジェクトの公開スペース（GitHub Issues, Pull Requests, Discussions など）および個人がプロジェクトを代表して活動するすべての場面に適用されます。

## 連絡先

問題が発生した場合は、GitHub Issues または プロジェクトのメンテナーに連絡してください。すべての報告は守秘義務のもとで審査されます。
