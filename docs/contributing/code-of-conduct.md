# 行動規範

## 目的

このプロジェクトのコミュニティを、すべての参加者にとって安全で歓迎される環境にすることを目的とします。

## 行動基準

**推奨される行動:**

- 他の参加者に対して親切で礼儀正しく接する
- 建設的なフィードバックを行い、受け入れる
- コミュニティ全体の利益を優先する
- 異なる視点・経験を尊重する

**禁止される行動:**

- ハラスメント、侮辱、差別的な言動
- 個人への攻撃・公私を問わず嫌がらせを行う行為
- 他者のプライバシーを侵害する行為（許可なく個人情報を公開するなど）
- その他、専門的な場にふさわしくない行為

## 責任範囲

プロジェクトメンテナーは、行動基準の解釈・実施に責任を持ちます。不適切な行動に対しては、コメントの削除・PR のクローズ・参加の禁止などの措置を取る場合があります。

## 適用範囲

この行動規範は、プロジェクトの GitHub リポジトリ（Issues、Pull Requests、Discussions、コメント）において適用されます。

## 連絡先

問題が発生した場合は、リポジトリの Issue または GitHub の通報機能を通じてメンテナーに連絡してください。
